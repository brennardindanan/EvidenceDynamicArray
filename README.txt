### Program's Purpose
	 To work with the dynamic array, EvidenceData, and be able to add to or delete evidence from the evidence collection, print data into an organized and readable chart, as well as perform initialization function, deep copy function, and growth/add/remove/print/cleanup functions. 
	
### List of Files
- defs.h: Contains source code needed to execute
- init.c: Contains source code needed to execute
- evidence.c: Contains source code needed to execute 
- main.c: Contains source code needed to execute
- README.txt: Contains this description

### Compiling and Executing
1. Navigate to the folder containing the source code in a terminal,
2. Use the command 'make' to create the file 'a3' which can be executed,
3. use the command './a3' while in the folder containing the executable to run the program

### Usage Instructions
1. After executing the file 'a3' from the steps above, observe the options in the MAIN MENU,
2. Choose to either
	a) Add evidence by pressing 1
		i) Enter the prompted information and follow the steps to add evidence to the collection
	b) Deleting evidence by pressing 2
		i) Follow the steps to delete the desired information from the evidence collection
	c) Printing evidence by pressing 3
		i) Observe the printed chart, made to be organized and readable. 
	d) Exiting the program by pressing 0
